# POSIX Reference

POSIX.1-2017 (IEEE Std 1003.1-2017) Shell and Utilities specification.

## Online reference

- Base URL: https://pubs.opengroup.org/onlinepubs/9699919799/utilities/
- Per-utility: `https://pubs.opengroup.org/onlinepubs/9699919799/utilities/<util>.html`
- License: © IEEE and The Open Group. All Rights Reserved.

Local copies of the spec are in `wiki/posix/` (gitignored — download with `scripts/download-posix-specs.sh`).
